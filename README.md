Criação do primeiro app com o curso do Android Studio

Quis cria-lo como o exemplo do curso mas com o meu toque de personalidade <br>

### O QUE APRENDI?

Aprendi a usar o @Compose (cria componente) <br>
Padding = Cria espaço entre a borda e a escrita<br>
.dp e .sp = são a mesma quantidade de espaço<br>
fontSize = tamanho da letra<br>
lineHeight = altura da tela<br>
TextAlign.Center = alinha texto no centro<br>
Alignment.End = alinha texto na lateral direita<br>
Row = alinha elementos lado a lado<br>
Column = alinha elementos de cima para baixo<br>
